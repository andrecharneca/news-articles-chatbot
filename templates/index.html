NewsGPT

A ChatGPT based retrieval-augmented QA system

This is an enhanced version of ChatGPT with access to a bank of news articles. It retrieves passages from the articles and uses them to answer questions or perform tasks.
The querying system is based on llama_index and the app is written in Python using Flask.
